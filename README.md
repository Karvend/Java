# 🚌 Bus Booking System

## 📌 Description
The **Bus Booking System** is a simple console-based Java application designed for a travel agency to handle passenger bus bookings. This project does not use a database and is completely file-independent — making it ideal for understanding core Java concepts. It is a minor project built purely using object-oriented programming principles.

This is also my **first project in Java**, aimed at building a strong foundation and creating something useful for small travel businesses or individual learning purposes.

---

## ✨ Features
- User-friendly and easy to understand
- Designed specifically for small-scale travel businesses
- Demonstrates core Java concepts like classes, objects, and method handling
- No external dependencies or database usage

---

## 🛠️ Technologies Used
- Java (Core)
- Visual Studio Code / Code::Blocks

---

## 📁 Folder Structure
BusBookingSystem/
├── Bus.java // Contains details and methods related to Bus objects
├── Booking.java // Manages booking-related data and logic
└── BusDemo.java // Main class that runs the program

---

## 🚀 How to Run

1. Open your preferred Java IDE (e.g., Visual Studio Code or Code::Blocks).
2. Create a new project and place the three files (`Bus.java`, `Booking.java`, and `BusDemo.java`) in the source folder.
3. Run `BusDemo.java` — this is the main class containing the `main()` method.
